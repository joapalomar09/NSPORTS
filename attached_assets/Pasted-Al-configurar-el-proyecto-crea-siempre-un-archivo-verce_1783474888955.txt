Al configurar el proyecto, crea siempre un archivo vercel.json en la raíz para asegurar el despliegue correcto de una SPA (Single Page Application) con Vite.
El archivo debe tener exactamente esta configuración:
framework: "vite"
outputDirectory: "dist/public"
rewrites: Una regla que redirija todas las rutas (source: "/(.*)") a /index.html para que el routing del lado del cliente funcione sin errores 404.
No esperes a que falle el deploy, inclúyelo desde el principio.

OBJETIVO DEL PROYECTO

Desarrollar una plataforma web premium para una agencia de representación de futbolistas profesionales y juveniles que transmita prestigio, confianza, innovación y proyección internacional.

La web debe posicionar a la agencia como una de las más importantes de Argentina y Latinoamérica, con una identidad visual propia, evitando copiar otros sitios. La experiencia debe ser comparable a una marca de lujo del deporte, con un enfoque cinematográfico, navegación fluida y excelente rendimiento.

El sitio debe estar diseñado para captar nuevos talentos, generar confianza en jugadores, familias y clubes, exhibir los futbolistas representados y servir como carta de presentación institucional.
